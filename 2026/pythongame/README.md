

# Разработка мультиплеерного платформера на Python

Этот проект представляет собой план и реализацию мультиплеерного платформера с архитектурой клиент-сервер.

## План разработки

### Этап 1: Подготовка
1. **Графический движок:** Arcade (использование OpenGL для производительности).
2. **Сетевой протокол:** TCP (надежная доставка данных через `socket` или `asyncio`).
3. **Архитектура:** Authoritative Server (сервер является единственным источником истины, обсчитывает всю физику и коллизии).
4. **Формат обмена данными:** JSON (простота отладки и универсальность).

### Этап 2: Прототип на стороне сервера
1. Реализация игрового цикла (Tick rate) на сервере.
2. Реализация физики платформера на сервере (гравитация, коллизии с тайлами).
3. Управление состоянием мира (координаты игроков, платформы).

### Этап 3: Сетевая часть (Сервер)
1. Создание TCP-сервера на сокетах.
2. Обработка подключений нескольких клиентов через `asyncio`.
3. Прием команд от клиентов (нажатия клавиш) в формате JSON.
4. Рассылка состояния мира (GameState) всем игрокам в формате JSON.

### Этап 4: Клиентская часть (Arcade)
1. Создание окна игры на библиотеке Arcade.
2. Отрисовка игрового мира на основе данных от сервера.
3. Отправка ввода игрока (клавиатура) на сервер.
4. Обработка сетевых задержек и отображение других игроков.

### Текущий прогресс
* **Этап 1: Подготовка** ✓ (Arcade установлен, TCP выбран, Authoritative Server реализован)
* **Этап 2: Прототип на стороне сервера** ✓ (Игровой цикл, физика, состояние мира в `server.py`)
* **Этап 3: Сетевая часть (Сервер)** ✓ (TCP на `asyncio`, JSON команды, рассылка состояния)
* **Этап 4: Клиентская часть (Arcade)** ✓ (Окно игры, отрисовка, отправка ввода в `client.py`)
* **Этап 5: Полировка и оптимизация** ✓ (Разноцветные платформы, небо с облаками)
* **Этап 6: Боевая механика** ✓ (Гранаты с физикой отскоков, трением, таймером взрыва, эффектами мощной взрывной волны, отбрасыванием игроков и других гранат, а также звуком взрыва)
* **Этап 7: Звуковое сопровождение** ✓ (Добавлены звуки ходьбы и прыжков)
* **Этап 8: Улучшенная визуализация персонажей** ✓ (Игроки теперь выглядят как человечки с головой, телом и анимированными конечностями)

### Инструкции по запуску
1. Установите зависимости: `pip install arcade`
2. Запустите сервер: `python main.py server`
3. Запустите одного или нескольких клиентов: `python main.py client`

## Технологический стек
* **Язык:** Python 3.10+
* **Библиотека графики:** Arcade
* **Сетевой протокол:** TCP (sockets / asyncio)
* **Архитектура:** Authoritative Server
* **Формат данных:** JSON
